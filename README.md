# OPUS

Персональный учебный трекер Арсения.

## Что внутри

- текущий урок;
- общий прогресс;
- дорожная карта обучения;
- уровни навыков;
- ближайшие задачи;
- журнал занятий;
- словарь терминов простым языком;
- экспорт и импорт прогресса в JSON.

Весь интерфейс находится в одном файле `index.html`. Данные сохраняются локально в браузере через `localStorage`.

## Запуск

Открой `index.html` в браузере.

## GitHub Pages

Чтобы открыть трекер как сайт:

1. Зайди в `Settings` репозитория.
2. Открой `Pages`.
3. В разделе `Build and deployment` выбери `Deploy from a branch`.
4. Выбери ветку `main` и папку `/root`.
5. Нажми `Save`.

После этого GitHub создаст публичную ссылку на приложение.

## Обновление прогресса

После занятия можно сохранить журнал в интерфейсе и экспортировать JSON. Сам HTML можно обновлять отдельными коммитами по мере развития программы обучения.
